`dired.nvim` aims to be a robust [dired](https://gnu.org/software/emacs/manual/html_node/emacs/Dired.html) implementation that tries to make file operations as fast as possible.

- Navigate: use `g` + `<key>` to open up the entry in two keystrokes.
- Create: use `o` and type `java/what/the/heck/Main.java<cr>` and the file will be there with your cursor on the entry.
- Delete: simply `dd` or `d` on selection
- No save, no confirmation, you know what you are doing (unless you don't, but you can always restore files from trash!).
